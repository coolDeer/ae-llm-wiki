# 智谱收入拆解、算力部署及三年展望（260411）

## 来源概要

对智谱 AI 经营数据的量化拆解，含 token 调用量反推收入、算力约束分析、与全球头部模型公司对比及三年收入天花板测算。注：数据来自个人搜集与专家调研，非官方。

## 关键要点

1. **核心数据**：文本 API 3300 亿 token/日，Coding Plan 1.4 万亿 token/日，合计日均吞吐约 4.7 万亿 token，算力已触顶。月 ARR 反算约合 2.5 亿美元（2026 年 3 月底）。
2. **关键判断**：三年后收入天花板 $15–35 亿（乐观），核心变量是算力获取能力；算力约束是最硬增长瓶颈，而非模型能力。
3. **行业参与者行为**：大客户占总用户 70–80%，典型客户包括天眼查/企查查（数据标签）、高德/小红书（安审，单客户日均百亿 token）、摄像头安防、Boss 直聘/探迹。云厂商合作显著收缩（火山因自身算力不足退回 1 亿+订单，阿里 2024 年无合作）。
4. **与共识差异**：GLM-5.1 在 SWE-bench 接近 Claude Opus 4.6 水平，但算力规模差距 30–50 倍，"核心差距不在模型，在算力"。与 OpenAI 100 万+ GPU、Anthropic 锁定 3.5 GW 相比，智谱仅 3 万张 A100 为主。
5. **时效性信号**：Coding Plan 护城河仅来自数据飞轮和迭代速度，"约一个半月就会有其他模型追平其代码能力"，护城河很浅。

- GLM-5 API 基准价：$1.0/M 输入 + $3.2/M 输出（vs Claude Opus 4.6：$5/$25）
- 海外节点：仅新加坡投用，马来西亚在建
- 2024 年 1 月被列入美国实体清单，无法采购 B200/H200

## 值得注意的观点/引语

> "算力规模差距是 30–50 倍，这不是效率能弥补的"

> "如果专注代码方向的厂商后续版本技术提升停滞，约一个半月左右就会有其他模型厂商追平其代码能力"

> "国产模型都是面临类似问题，都只是以 10%–20% 的价格做海外模型的平替，彼此之间没法作出性能的明显差异"

## 结构性观察

- 智谱的商业模式本质是"低成本国产替代"——以政府算力补贴（40–50% 划拨算力）换取价格竞争力，以国家队背景换取政府订单。这种模式可持续，但无法自主定价，极其脆弱：一旦英伟达卡解禁，竞争力消失。
- 海外业务是政策驱动而非商业驱动（仅东南亚/中东），这与 Anthropic 自主全球拓展形成鲜明对比。
- Coding Plan 的数据飞轮优势是当前差异化的主要来源，但这种飞轮效应在算力饱和时会被耗尽，而算力扩张渠道已被实体清单堵死。
