# BofA 建设银行 FY25 业绩交流纪要

> 来源：BofA Securities Post Result Call | 日期：2026-04-10
> 公司：[[company/建设银行|建设银行]]（601939.SH / 0939.HK）

## 核心要点

### 2026年信贷增长

- 全年新增贷款总量预计与2025年大致持平
- 增速较2025年的约8%下降约1个百分点，但仍跑赢市场整体
- 信贷投放策略从规模驱动转向质量优先

### 零售贷款

- Q1零售整体增长缓慢，约1%
- 按揭贷款继续负增长，拖累零售表现
- 消费贷款和个人经营性贷款增速快于整体，结构性亮点

### 存款

- 全年增长目标约8%
- 个人存款增长超过8%
- 存款结构出现显著变化：长期限存款向短期限存款迁移
- 管理层判断居民正在为[[industry/中国银行业|资本市场]]机会做准备

### NIM（净息差）

- 2025年下降部分由于信用卡收入一次性入账调整
- NIM水平在国有大行中仍处于高位
- 2026年NIM降幅预计收窄
- 贷款端重定价基本完成
- 负债端成本持续改善

### 中间业务收入（Fee Income）

- 2025年受WMP（理财产品）重新并表影响，一次性减少数十亿元
- 2026年Q1对公业务表现最强
- 消费代理费收入实现双位数增长

### 债券策略

- 持续增加债券配置
- 收益率曲线陡峭化令net interest spread优势扩大
- 交易性收入预计较2025年"有所减少"

### 资产质量

- 零售NPL上升速度趋缓
- 房地产开发贷NPL约5%，已在该水平波动约2年，大幅上升的可能性较低
- 继续主动处置不良资产

### 2026年经营目标

- 收入和利润增长好于2024/2025年
- 净利息收入在连续三年下降后，预计实现较好增长

|------|------|

| 贷款增速 | ~7%（较2025年~8%降约1pp） |

| 存款增长目标 | ~8% |
| 个人存款增长 | >8% |
| 房地产开发贷NPL | ~5% |

## 投资启示

1. NIM压力最大的阶段可能已过，2026年拐点初现
2. 零售端短期承压但消费贷/经营贷结构优化在进行中
3. 存款搬家趋势值得关注——若居民资金流入资本市场，可能影响低成本存款基础
4. Fee income的WMP并表一次性影响消除后，2026年基数效应有利

## 结构性观察

- **信贷增长从"规模驱动"转向"质量优先"，反映国有大行战略意图的根本性转变**：这不是市场低迷导致的被动收缩，而是管理层主动选择——意味着 NIM 托底逻辑的前提是贷款结构改善，而非单纯依赖规模效应。这一转变若全行业同步，将对银行股估值体系产生长期影响。
- **居民存款从长期向短期迁移，是"资本市场预备金"的行为信号**：管理层将此解读为居民为权益市场机会做准备，若这一判断正确，则意味着低成本存款基础将在牛市中承压——这是银行负债端最难对冲的结构性风险，也是评估建设银行 NIM 持续性的关键变量。
- **房地产开发贷 NPL 在约 5% 水平波动约 2 年，表明银行体系已主动消化了大部分不良**：不良形成的尾部风险边际下降，但这并不等于地产风险完全出清——存量处置进度将持续影响拨备压力，是分析银行利润弹性的核心变量。
