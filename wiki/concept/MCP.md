# MCP (Model Context Protocol)

## 定义

MCP（Model Context Protocol，模型上下文协议）是由 [[company/Anthropic|Anthropic]] 推出的开放协议，旨在标准化大语言模型（LLM）与外部工具、数据源之间的交互接口。通过 MCP，企业可以将 AI 模型无缝连接到现有的业务系统和数据管道中，大幅降低集成复杂度。

## 在投资研究中的应用

### 企业集成的关键基础设施

MCP 在 [[company/Anthropic|Anthropic]] 的企业战略中扮演核心角色：

- **降低集成复杂度 80%**：企业部署 AI 的最大障碍之一是与现有系统的集成。MCP 通过标准化协议将这一复杂度降低 80%。
- **支撑 "Land and Expand" 策略**：MCP 降低了企业试用门槛，使 PLG（Product-Led Growth）策略更易执行。
- **合作伙伴生态的技术基础**：$100M 合作伙伴投资和 AABG 30,000 人团队的服务能力建立在 MCP 标准化接口之上。

（来源：[[source/Merit-Anthropic企业业务调研-260410|Merit-Anthropic企业业务调研-260410]]）

### 投资含义

- MCP 的广泛采用可能成为 [[company/Anthropic|Anthropic]] 生态系统的 "护城河"——一旦企业基于 MCP 完成集成，切换成本显著提升
- 作为开放协议，MCP 也可能被竞争对手采用，成为行业标准而非 Anthropic 独有优势
- MCP 生态的成熟度直接影响企业 AI 部署速度和 Anthropic 的收入增长
