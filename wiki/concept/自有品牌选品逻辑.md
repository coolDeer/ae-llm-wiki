## 定义

自有品牌（Private Label / Store Brand）选品逻辑是指零售商判断哪些品类/SKU适合开发自有品牌的系统化决策框架，核心目标是在提升品牌差异化和顾客忠诚度的同时，实现显著高于外部品牌的综合毛利率。

## 在投资研究中的应用

### 山姆/沃尔玛选品标准（四维评估框架）

1. **品类成熟度**：该品类在中国零售市场的渗透率、市场规模、销售表现（需通过第三方合法数据验证）
2. **子品类适用性**：在具体SKU维度（如"牛奶口味夹心饼干" vs "柠檬口味"）确认最优切入点
3. **渠道/顾客需求匹配**：本渠道客群对该品类的需求强度和认可度
4. **毛利空间**：自有品牌综合毛利率须显著优于对标商品（否则开发意义不大）

（来源：[[source/cb-商超自有品牌-260412]]）

### 山姆 vs 沃尔玛的选品差异

| 维度 | 山姆Member's Mark | 沃尔玛沃集鲜/惠宜 |
|------|-----------------|----------------|
| 品质标准 | 90-100分（高品质）| 80-90分 |
| 价格定位 | 80-85分（高性价比）| 70-75分 |
| 选品范围 | 刚需高频 + 差异化特色品类 | 仅高频刚需成熟品类 |
| 创新能力 | 冷藏小青柠饮料等爆品 | 保守，不做新奇特品类 |
| 战略目标 | 提升会员续费率（会员费=核心收入）| 提升销量和毛利 |

### 开发流程耗时分布（总周期约1年）

| 阶段 | 耗时 |
|------|------|
| 立项/数据分析 | 2-6个月 |
| 找供应商+样品验证 | 1-2个月 |
| 方案提交/审批 | 1-2个月 |
| 生产/上架 | 2-3个月 |

### 定价策略

- 自有品牌定价：比对标商品低10%以上
- 参考维度：对标商品价格 + 其他渠道同类价格

## 品类适配案例

| 品类 | 适合做自有品牌 | 原因 |
|------|-------------|------|
| 粮油/纸巾/牛奶 | 是 | 成熟品类，高频刚需，毛利空间可预测 |
| 冷藏小青柠饮料 | 是（创新）| 新品类空白，山姆客群有需求且敢于尝新 |
| 有机高端食材 | 山姆是，沃尔玛否 | 需要匹配客群消费能力 |
| 预制菜 | 暂不重点 | 沃尔玛/山姆当前未列为自有品牌重点 |
