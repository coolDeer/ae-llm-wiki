## 定义

美国-伊朗关系是中东地缘政治的核心双边关系，对全球能源供应（霍尔木兹海峡）、地区安全格局（伊核协议）及大宗商品定价具有系统性影响。2026年Q1因军事冲突急剧恶化后，4月进入停火-谈判阶段（来源：[[source/vk-US-Iran-Pakistan-260412]]）。

## 在投资研究中的应用

### 宏观资产价格传导链

```
美伊冲突升级 → 霍尔木兹封锁 → 油价飙升 → 美元走强（能源独立优势）→ 利率上行 → 股市承压

美伊缓和 → 霍尔木兹开放 → 油价下行 → 美元走弱 → 利率下行 → 股市受益 + EM相对跑赢
```

（来源：[[source/substack-宏观框架更新-260412]]，Macro Charts核心框架）

### 关键谈判要点（2026年4月）

**核心分歧（截至4月12日）**：
- 霍尔木兹控制权
- 现有浓缩铀库存处置
- 伊朗的铀浓缩权利

**格局评估**：
- 美国升级选项均代价极高（武力重开霍尔木兹需数十万地面部队）→ 升级不太可能
- 伊朗维持封锁成本极低 → 伊朗谈判筹码强
- 停火期限4月21日，两国均未排除后续谈判

### 历史参照

- 2015 JCPOA（奥巴马核协议）：谈判耗时2年
- 伊朗曾承诺不寻求核武器，再次承诺并非不可能
