## 定义

美国-伊朗关系是中东地缘政治的核心双边关系，对全球能源供应（霍尔木兹海峡）、地区安全格局（伊核协议）及大宗商品定价具有系统性影响。2026年Q1因军事冲突急剧恶化后，4月进入停火-谈判阶段（来源：[[source/vk-US-Iran-Pakistan-260412]]）。

## 在投资研究中的应用

### 宏观资产价格传导链

```
美伊冲突升级 → 霍尔木兹封锁 → 油价飙升 → 美元走强（能源独立优势）→ 利率上行 → 股市承压

美伊缓和 → 霍尔木兹开放 → 油价下行 → 美元走弱 → 利率下行 → 股市受益 + EM相对跑赢
```

（来源：[[source/substack-宏观框架更新-260412]]，Macro Charts核心框架）

### 关键谈判要点（2026年4月）

**核心分歧（截至4月12日）**：
- 霍尔木兹控制权
- 现有浓缩铀库存处置
- 伊朗的铀浓缩权利

**格局评估**：
- 美国升级选项均代价极高（武力重开霍尔木兹需数十万地面部队）→ 升级不太可能
- 伊朗维持封锁成本极低 → 伊朗谈判筹码强
- 停火期限4月21日，两国均未排除后续谈判

### 4月13日更新：霍尔木兹封锁范围澄清

（来源：[[source/vk-US-Iran-followup-260413]]，2026-04-13）

- **封锁范围窄于初看**：五角大楼声明澄清，仅针对进出伊朗港口和海岸区域船只，允许其他船只通行——并非全面封堵所有交通。
- **特朗普不太可能恢复全面轰炸**：WSJ 报道及五角大楼信号均指向特朗普远离激烈军事行动路线，地面部队部署概率更低。
- **巴基斯坦会谈结果**：未达整体协议但产生"善意氛围"；白宫认为伊朗"仍可能接受条件"；伊朗外交部表态继续谈判。
- **VK 核心判断维持**：停火将持续（ceasefire will hold）；"特朗普被困"（无法升级、无法退出）是维持低烈度状态的结构性逻辑。
