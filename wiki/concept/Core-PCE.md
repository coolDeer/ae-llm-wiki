# Core PCE（核心个人消费支出价格指数）

## 定义

Core PCE（Core Personal Consumption Expenditures Price Index，核心个人消费支出价格指数）是剔除食品和能源后的个人消费支出价格指数，由美国商务部经济分析局（BEA）按月发布。它是 Federal Reserve 首选的通胀衡量指标，因其覆盖范围更广、权重动态调整（链式加权），被认为比 CPI 更准确地反映实际消费价格变化。

## 与 CPI 的区别

| 维度 | Core PCE | Core CPI |
|------|----------|----------|
| 发布机构 | BEA（商务部） | BLS（劳工部） |
| 权重调整 | 链式加权（动态调整） | 固定篮子（定期更新） |
| 覆盖范围 | 更广，含第三方支付（如雇主医保） | 仅消费者直接支出 |
| 住房权重 | 较低（~15%） | 较高（~33%） |
| Fed 参考 | 首选指标 | 辅助参考 |
| 波动性 | 通常更低 | 通常更高 |

## 在投资研究中的应用

### 当前状况（2026 年 3 月）

基于[[source/BofA-美国3月CPI快评-260410|BofA 3 月 CPI 快评]]的最新数据：

- **Core PCE MoM 推算**：+0.20%（基于 3 月 Core CPI +0.20% 推导）
- **Core PCE YoY 预测**：从当前 3.0% 升至年底 3.1%
- **上修原因**：油价上涨 30%（伊朗战争）的间接传导（机票 +2.7%、配送 +3.1%）+ 基数效应
- **此前预测**：2.8%，因能源传导效应上修 30bp 至 3.1%

### 投资含义

1. **Fed 政策锚定**：Core PCE 是 Fed 设定利率的核心参考。当前 3.0-3.1% 水平仍高于 2% 目标，但下行趋势支持降息预期。
2. **CPI-PCE 转换**：由于权重差异，Core CPI 数据需通过模型转换为 Core PCE 才能准确评估 Fed 反应函数。例如，3 月 Core CPI +0.20% 对应 Core PCE 约 +0.20%。
3. **能源传导时滞**：油价对 Core PCE 的影响有 1-3 个月时滞，通过机票、配送等间接渠道传导。30% 的油价上涨预计推动 Core PCE 年底上行约 30bp。
4. **通胀预期与实际通胀的分离**：UMich 1 年期通胀预期跳升至 4.8%，但 Core PCE 仍在 3% 附近，两者分化值得关注——若预期持续上行可能影响 Fed 决策。

（来源：[[source/BofA-美国3月CPI快评-260410|BofA 美国 3 月 CPI 快评]]）

## 数据发布

- **频率**：月度
- **发布时间**：通常在数据月份后约 4 周（比 CPI 晚约 2 周）
- **发布机构**：Bureau of Economic Analysis (BEA)
- **市场关注度**：极高，尤其在 FOMC 会议前后
