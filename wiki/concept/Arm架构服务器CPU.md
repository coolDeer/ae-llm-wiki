# Arm 架构服务器 CPU

## 定义

Arm 架构服务器 CPU 是采用 ARM 指令集架构的服务器处理器，以低功耗、高效率著称，近年在 AI 推理场景和信创市场快速渗透，对 x86（Intel/AMD）垄断地位形成挑战。

（来源：[[source/ace-CPU与GPU供需及国内格局-260413]]）

## 在投资研究中的应用

### 全球服务器 CPU 市场份额（2025 年底）

| 架构 | 份额 |
|------|------|
| Intel（x86） | 60% |
| AMD（x86） | 24% |
| Arm 架构 | 16% |

（来源：[[source/ace-CPU与GPU供需及国内格局-260413]]）

### Arm 重返服务器市场的路径演进

- **此前失败**：因生态不兼容（非 x86 生态）失败
- **现在的切入逻辑**：AI 推理场景（CPU 负担不高）+ AI Agent 爆发重新切入
- **市场定位转变**：从"取代 x86"的正面竞争 → "填充 GPU+CPU 推理架构中 CPU 侧的增量需求"

> "Arm 将自身精准定位于 AI Agent 和推理前端，认为这是重返服务器市场的良机。相较于已接近饱和的边缘计算市场，服务器领域是其寻求增长的唯一方向。"

（来源：[[source/ace-TPU出货更新与AI算力动态-260413]]）

### Intel CPU 涨价信号（2026 年 5 月）

- Intel 将于 2026 年 5 月针对 64 核及以下处理器进行结构性涨价（供不应求最严重品类）
- 涨价反映 AI 推理对中低核数服务器 CPU 的超预期需求弹性（边缘推理、智能体调度等新场景）

（来源：[[source/ace-TPU出货更新与AI算力动态-260413]]）

### 中国特殊性

- **信创政策**驱动：中国 Arm 渗透率高于全球平均（华为鲲鹏 + 飞腾推动）
- 中国 PC 出货量约 4,000 万台，信创峰值约 600–800 万台
- 需区分：中国 Arm 渗透是政策驱动，与海外 Arm 凭推理效率抢夺服务器份额逻辑不同

（来源：[[source/ace-CPU与GPU供需及国内格局-260413]]）

## 结构性观察

- Arm 架构的渗透路径已从"取代 x86"转向"填充 GPU+CPU 推理架构中 CPU 侧的增量需求"——这是更现实但也更可持续的增长逻辑
- Intel CPU 涨价反映的是推理场景的新增需求（Agent/边缘推理），而非传统服务器市场恢复
