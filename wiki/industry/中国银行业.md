# 中国银行业

## 行业概况

中国银行业是全球最大的银行体系，总资产规模超过400万亿元人民币。行业由国有大行主导，包括工商银行、[[company/建设银行|建设银行]]、农业银行、中国银行、交通银行和[[company/邮储银行|邮储银行]]六大国有银行，以及股份制银行、城商行、农商行等多层次银行体系。

## 行业格局

### 国有大行（Big Six）

| 银行 | 特征 |
|------|------|
| 工商银行 | 资产规模最大，综合实力最强 |
| [[company/建设银行\|建设银行]] | 基建/住房金融传统优势，NIM大行领先 |
| 农业银行 | "三农"业务优势，县域覆盖广 |
| 中国银行 | 国际化程度最高，外汇业务领先 |
| 交通银行 | 规模相对较小，正推进深化改革 |
| [[company/邮储银行\|邮储银行]] | 网点最多，存款成本最低，成长性最强 |

### 竞争态势

- 国有大行占据约40%市场份额，具有系统重要性
- 股份制银行在零售、财富管理等领域形成差异化竞争
- 城商行/农商行深耕区域市场

## 行业趋势（2025-2026）

基于[[source/BofA-建设银行FY25业绩交流-260410|建设银行]]和[[source/BofA-邮储银行FY25业绩交流-260410|邮储银行]]FY25业绩交流的行业信号：

### 信贷增长

- 行业整体信贷增速放缓，大行增速从~8%降至~7%区间
- 信贷投放从"量"转向"质"成为共识
- 新增贷款总量与上年持平，结构优化是重点

### NIM（净息差）

- 行业NIM持续下行但降幅收窄
- 贷款端重定价基本完成（[[company/建设银行|建设银行]]）
- 存款成本改善提供对冲（[[company/邮储银行|邮储银行]]存款成本降至~1.0%）
- 对公贷款利率出现企稳迹象（邮储对公利率回升至>3%）

### 零售银行

- 按揭贷款普遍负增长或低增长
- 消费贷款和个人经营性贷款成为新增长点
- 零售NPL整体上升但速度趋缓
- 居民存款出现从长期限向短期限迁移的趋势

### 非利息收入

- Fee income成为重要增长引擎
- 财富管理、基金代销等代理业务增长强劲
- 投行和交易银行业务高速增长
- 债券投资和交易收入贡献增加

### 资产质量

- 房地产开发贷NPL趋于稳定（建行~5%，波动约2年）
- 零售NPL上行但速度放缓
- 小微贷款资产质量分化——邮储改善显著（90+逾期从>2%降至~1%）
- 银行继续主动处置不良资产

### 资本与监管

- 国家资本注入增强大行资本实力（邮储获1,300亿元注入）
- AIRB等高级方法推广有望释放资本空间
- 分红率保持稳定（大行普遍30%左右）

## 投资框架

### 关注要素

1. NIM拐点时机——降幅收窄后何时企稳
2. 资产质量趋势——房地产和零售NPL演变
3. Non-interest income增长持续性
4. 资本补充与分红空间
5. 宏观经济复苏强度

### 行业主题

- **NIM见底**：贷款重定价完成 + 存款成本改善 = NIM降幅持续收窄
- **收入多元化**：fee income和投资收益占比提升
- **质量优先**：从信贷规模扩张转向资产质量和收益率优化
- **资本强化**：国家注资 + AIRB转型释放资本

## 宏观视角：地产信用与资本流动（更新：2026-04-13）

（来源：[[source/mm-GS中国宏观地产资金流-260413]]）

- **港股地产 vs 内地地产分化**：HK 住宅价格预计 +10%，内地地产仍在去库存（约 -40% 去化压力）。银行对内地地产贷款敞口仍承压，但香港银行受益于港股地产回暖
- **十五五规划"109 项目"**：预计 GDP 增量约 3 万亿元，是信贷需求的政策底。银行信贷主要流向基建/制造业，而非地产
- **PPI 拐点与信贷质量**：PPI 从负转正将改善国内生产企业现金流，降低银行不良率
- **资本偏好极化**：顶尖白领青睐 A 股/港股/大宗商品，散户资金延续"保守+避险"格局——这对国有大行零售存款成本影响有限，但对小银行存款竞争有一定压力
