# CIS（CMOS图像传感器）行业

## 行业概况

CMOS图像传感器（CIS，CMOS Image Sensor）将光信号转化为电信号，是手机摄像、汽车ADAS、工业机器视觉、安防监控等领域的核心芯片。按技术架构分为：
- **Rolling Shutter（卷帘快门）**：主流，用于手机/汽车/安防
- **Global Shutter（全局快门）**：工业/科研领域主流，国内[[company/长光辰芯]]擅长

## 市场规模与增长

| 市场 | 规模 | 增速CAGR | 特点 |
|------|------|---------|------|
| 手机CIS | 全球~150亿美元（含占比50%+） | 个位数 | 高端绑定，低端分散 |
| 汽车CIS | 增长中 | >20%（最高） | 单车量增；国产渗透初期 |
| 工业CIS（国内） | <5亿美元 | ~15% | Global Shutter主导，格局固定 |
| 安防CIS | 相对稳定 | <10% | 国产化率高 |

（来源：[[source/cb-CIS-长光辰芯-260412]]）

## 手机CIS竞争格局

### 全球格局（出货量/营收）
| 厂商 | 优势细分 | 份额 |
|------|---------|------|
| 索尼（Sony） | 大像素50MP（1.2-1.6μm） | 全球第一（含约40%+在大像素领域） |
| 三星（Samsung） | 小像素200MP（0.5-0.7μm）；200MP长焦独家 | 全球第二（小像素领先） |
| 豪威（OV/OMNIVISION） | 中端综合 | 第三梯队 |
| SmartSens（思特威） | 国内特色，华为主供 | 国内快速增长 |
| 格科微 | 低端市场性价比 | 单芯片方案抢份额 |

### 国内品牌切换进度（2025年）
| 手机厂商 | 状态 |
|---------|------|
| 华为 | 几乎全系SmartSens（华为因制裁无法使用索尼） |
| 小米 | 数字系列15起主摄切国产；广角/潜望仍用三星 |
| 荣耀 | 多用国产 |
| OPPO/vivo旗舰（4,500+元） | **仍绑定索尼/三星**（定制/排他合作） |
| 子品牌（红米/真我/OnePlus等） | 大量使用格科微/元视芯 |

**索尼/三星短期份额稳定**：50MP大像素索尼领先1-2代；200MP长焦三星独家；国内差距约5年内难以消除。（来源：[[source/cb-CIS-长光辰芯-260412]]）

## 工业CIS竞争格局

工业CIS = Global Shutter为主，市场极度集中：

| 厂商 | 优势 |
|------|------|
| 安森美（PYTHON系列） | 中低端工业主导（合计>50%市占） |
| 索尼（IMX系列） | 中端工业+标准应用 |
| [[company/长光辰芯]] | 高端（4,000万+像素、线扫）国内第一梯队；近年下沉中低端 |
| Teledyne | 高端科学级 |
| CMOSIS（比利时） | 特殊应用 |

**海外设备OEM绑定极强**：Keyence清一色索尼，Cognex以索尼/安森美为主，切换难度极高。

## 长光辰芯深度分析

**背景**：长春光机所孵化，科研/军工背景，港股IPO中  
**定位**：高端Global Shutter起家→向中低端工业下沉  
**财务**：毛利率约40%+（高端为主）  
**流片**：主要依赖海外（高塔、东部高科DB），台积电路线较少  
**生态**：与宇勘（ISP芯片）配套，在国内科研/军工单位形成封闭生态

**下沉挑战**：中低端工业CIS（200万-500万像素）以安森美老款产品为主，价格竞争力强；长光需将毛利率从40%压缩至25-30%才能参与竞争

**IPO后战略（预判三步走）**：  
1. 继续下沉低端Global Shutter（稳固基本盘）  
2. 丰富工业产品线roadmap  
3. 消费级（无人机→汽车，优先汽车；手机最后考虑）

（来源：[[source/cb-CIS-长光辰芯-260412]]）

## 关键趋势

1. **国内手机CIS国产化加速但OPPO/vivo旗舰是"最后堡垒"**
2. **工业CIS国产渗透有空间但需时间**（长光工业份额2-3年内可能从<10%到15%）
3. **汽车CIS是最高增速赛道**（>20% CAGR），国内厂商汽车布局刚起步，认证周期约1年
4. **海力士退出手机CIS**：2024年H1战略退出，间接反映国产切换速度
