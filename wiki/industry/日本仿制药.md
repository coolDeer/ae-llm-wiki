# 日本仿制药

> Japanese Generic Pharmaceuticals

## 行业概况

日本仿制药行业正经历结构性整合。厚生劳动省推行的 A/B/C 稳定供应评级体系正在重塑竞争格局，订单向 A 级评级企业集中，中小企业面临淘汰压力。行业同时面临生产质量管控挑战、汇率驱动的 API（原料药）成本上升及药价政策压力。

---

## 行业结构

### 稳定供应评级体系（A/B/C）
- 厚生劳动省建立的质量/供应能力评级
- **A 级**：稳定供应能力最强，获得最多订单
- **B/C 级**：订单减少，面临被整合风险
- 推动行业订单向头部企业集中

### 行业整合趋势
- A 级企业订单量高企（如 [[沢井集团]] 无法全部承接）
- 中小企业产能退出
- 品牌药企出售工厂，为仿制药企提供收购机会
- 产能规模化成为核心竞争壁垒

---

## 成本结构

### API（原料药）成本
- 日元贬值（159 JPY/USD）推高进口 API 成本
- API 多依赖海外采购，汇率敏感度高

### 药价政策
- 中期药价调整对低价产品利润挤压明显
- 行业倡议：NHI 价格 ≤ ¥15 的产品免于中期调整
- 药价政策走向是影响行业盈利的关键变量

---

## 生产挑战

- 新产品上市延迟
- 程序性审批延误
- 新员工质量管控问题
- 部分工厂管理问题（如 [[沢井集团]] 三田工厂）

---

## 产能扩张策略

- 行业主流策略：**收购现有工厂**优于新建
- 品牌药企退出部分生产线，提供收购标的
- [[沢井集团]] 中期目标：产能 220 亿片，销售目标 190 亿片

---

## 关键驱动因素

1. **稳定供应评级** → 订单集中化，头部受益
2. **药价政策** → 直接影响收入和利润率
3. **汇率（JPY/USD）** → API 成本核心变量
4. **产能规模** → 规模效应决定竞争力
5. **仿制药替代率** → 政策推动下的结构性增长

## 风险因素

- 药价政策进一步收紧
- 日元持续贬值推高 API 成本
- 生产质量事故
- 产能扩张不及预期

## 催化剂

- 药价政策改善（低价产品免调整）
- 行业整合加速，头部市场份额提升
- 汇率回升降低 API 成本
- 亏损产品退出改善行业利润率
