# 沢井集团 (Sawai Group Holdings)

> Ticker: 4887.T | 交易所: TSE Prime | 行业: [[日本仿制药]]

## 公司概况

沢井集团是日本领先的仿制药企业，主营仿制药的研发、生产与销售。公司在厚生劳动省稳定供应评级体系中获得 A 级评价，受益于行业整合趋势。

## 商业模式

- 仿制药研发、生产与销售
- 受益于日本医疗费用控制政策下的仿制药替代趋势
- 产能规模化为核心竞争力
- 中期目标：产能 220 亿片，销售目标 190 亿片

## 财务摘要

| 指标 | 数据 |
|------|------|
| FY2026 展望 | 生产趋于稳定 |
| 汇率假设 | 159 JPY/USD |
| 亏损产品停产节省 | 约 ¥40 亿 |
| 产能目标 | 220 亿片 |
| 销售目标 | 190 亿片 |

- API（原料药）成本因日元贬值（159 JPY/USD）而上升
- 计划停产 100+ 亏损产品，预计减亏约 ¥40 亿

## 竞争格局

- 厚生劳动省 A/B/C 稳定供应评级体系正在重塑行业格局
- A 级评级企业获得更多订单，推动行业整合
- 沢井集团作为 A 级企业，订单量高企但受产能限制无法全部承接
- 行业趋势：中小企业被淘汰，订单向头部集中

## 风险因素

- **生产风险**：新产品上市延迟、程序性延误、新员工质量管控问题
- **三田工厂管理问题**：已更换领导层，但整改效果有待观察
- **汇率风险**：日元贬值推高 API 成本（159 JPY/USD）
- **药价政策**：中期药价调整可能压缩利润
- **产能瓶颈**：订单无法全部承接，制约收入增长

## 催化剂

- **亏损产品停产**：100+ 产品停产可节省约 ¥40 亿，直接改善利润率
- **产能扩张**：通过收购品牌药企工厂扩大产能（优于新建）
- **行业整合**：A 级评级推动订单集中，市场份额持续提升
- **三田工厂管理改善**：新领导层推动运营效率提升
- **药价政策倡导**：NHI 价格 ≤ ¥15 产品免于中期调整若获批，保护低价产品盈利
