# OpenAI

## 公司概况

OpenAI 是全球领先的 AI 研究和部署公司，核心产品为 ChatGPT（消费者端）和 API（企业端）。通过"梯云纵"资本结构与英伟达、微软、甲骨文深度绑定，以最少自有资金获取最多算力资源。

## 商业模式

### 收入来源

- **ChatGPT 订阅**（消费者端）：Free/Plus/Pro 分层
- **API 服务**（企业端）：按 token 计费
- **未来广告收入**：2030 年预计 $1020 亿（需 2.75B WAU，市场存疑）

### 梯云纵资本结构

OpenAI 主导的循环嵌套资本架构：

1. **英伟达** 早期注资 $1000 亿（以算力换股权，锁定 5–10 年 GPU 订单）
2. **微软** 提供云服务 + 收取 40% 营收分成（2024 年 $14 亿）+ 将模型嵌入 Office/Bing
3. **甲骨文** 签订 $3000 亿云服务协议，向英伟达采购 $1000 亿+ 芯片，英伟达回流 $1000 亿投 OpenAI
4. **软银** 等外部资本介入，形成多层"俄罗斯套娃"

本质：用未来预期透支当下资本，真正新增现金流有限。

（来源：[[source/cb-OpenAI梯云纵生态布局-260412]]）

## 竞争格局

### 用户数据

- ChatGPT 月活用户：9.61 亿（2026-03）
- 美国 MAU：连续 6 个月基本持平（面临 Gemini + Claude 竞争）
- 单用户年化价值：~$26（vs Claude 单用户 $808）

### 战略困境

同时在 C 端（vs Meta/Google）和 B 端（vs Anthropic）两线作战，算力约束下顾此失彼。

- C 端：用户增长最困难阶段已过，后续 5 亿 WAU 获取成本将大幅上升
- B 端：企业高价值用户被 Anthropic 主导（B 端市占 ~45% vs OpenAI ~25%）

（来源：[[source/substack-Meta-Muse-Spark-260412]]，[[source/cb-OpenAI梯云纵生态布局-260412]]）

## 产品矩阵

- **ChatGPT**：全球最大 AI 消费者产品，月活 9.61 亿
- **GPT API**：多版本模型（GPT-5 等），开发者生态；Agent 能力极强（ChatGPT 内置多模态+用户记忆，可直接构建自定义 Agent）
- **ChatGPT Atlas 浏览器**（2025 年 10 月推出，macOS 先行）：AI 原生浏览器，强调 Agent 能力、用户记忆、网页内操作自动化；竞争 Chrome/Edge（来源：[[source/cb-OpenAI-Atlas浏览器-260413]]）
- **Sora**（视频生成）
- **Stargate**：与软银/甲骨文合资，2029 年前投资 $5000 亿 AI 基础设施

## 独立化战略

> 更新（2026-04-13）：OpenAI 正在系统性构建与多方云厂/芯片商的独立合作关系。（来源：[[source/cb-AWS-vs-Azure-vs-GCP-260413]]）

- 与 AWS 合作使用 Trainium 芯片，试图摆脱对微软 Azure 和英伟达的单一依赖
- 定位为"AI 时代的独立合作人"——同时从微软、亚马逊、软银、英伟达获取资源，不被任何一家独占绑定
- 将自身定位为"公有云的竞争对手"，长期或推出 OpenAI Cloud 自有服务

## 风险因素

- 持续亏损，高度依赖外部融资（每季度需 $200 亿）
- 资本结构高度循环，依赖英伟达/微软/甲骨文三方绑定
- C 端用户增长见顶压力（美国 MAU 停滞）
- B 端被 Anthropic 超越（企业市占 25% vs 45%）
- 微软的营收分成（40%）持续挤压利润空间

## 催化剂

- Stargate 算力规模释放后的模型能力跃升
- 广告收入商业化（2026+ 有广告收入预期）
- 企业端反攻（新产品或定价策略调整）
