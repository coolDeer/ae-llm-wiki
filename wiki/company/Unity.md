# Unity Software

## 公司概况

Unity 是全球领先的游戏开发引擎提供商，同时运营移动广告网络业务（Unity Ads + IronSource）。近年来通过 UDA 算法强化广告业务，但整体增速低于早期预期。

（来源：[[source/mm-Merit-Applovin-Unity-广告-260413]]）

## 广告业务表现（2026 Q1）

| 指标 | 数据 |
|------|------|
| Q1 YoY 增速 | ~17-19%（低于早期预期 35%） |
| 预算份额（游戏广告主）| 8% |
| ROI 水平 | ~2.4（与 AppLovin 相当） |
| CPM/ROI QoQ 增速 | 2-3% |
| CPM/ROI YoY 增速 | >10% |

### 2026 Q2 全年预测

- Q2 YoY：>10%（QoQ +3-5%）
- 全年 YoY：15-20%

## 关键产品

### UDA 算法

- 广告主评价"相当不错"，效果稳定
- 部分广告主为早期大规模使用者

### IronSource（计划剥离）

- 当前广告主仍大量使用，ROI 稳定
- Unity 计划将 IronSource 剥离并将变现部分迁移至 LevelPlay
- 广告主当前认为风险不大，继续监控

## 竞争格局

- **vs Meta/Google**（一线平台）：ROI 劣势明显（2.4 vs 2.4-2.8）
- **vs AppLovin**：ROI 相当，但用户覆盖和生态有差异，广告主同时使用两者
- 游戏广告市场二线平台（与 AppLovin 同级）
